## Digital Bank user stories ##

This document shows the User Stories of this ***digitalbank-gen-one*** test automation project. This project is for practicing test automaition studies in a project work by Masterfield test automation training.
* US1: Login feature. As a user, I want to enter to the https://eng.digitalbank.masterfield.hu/bank/login page with a valid tester01 username and password so that I land to Dashboard on the /bank/home page.



* US2: Login fail feature. As a user, I want to enter to https://eng.digitalbank.masterfield.hu/bank/login page with an invalid tester username and password so that I can see the unsuccessful login message: "Invalid credentials or access not granted due to user account status or an existing user session."


* US3: Update Profile feature. As a user, I want to chang the first name in MyProfile setup so that it is visible in the Welcome text.


* US4: Deposit transaction feature. As a user, I want to transfer a new deposit amount so that it is visible in the checking list and it changes the balance.


* US5: Withdraw transaction feature. As a user, I want to transfer a new withdraw amount so that it is visible in the savings list and it changed the balance.
