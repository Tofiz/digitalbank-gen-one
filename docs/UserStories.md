## Digital Bank user stories ##

This document shows the User Stories of this ***digitalbank-gen-one*** test automation project. This project is for practicing test automaition studies in a project work by Masterfield test automation training.
* US1: Test login / logout. Verify the entering to https://eng.digitalbank.masterfield.hu/bank/login page with tester01 user lands to Dashboard on the /bank/home page. Click on the user-avatar picture and log out to get back to the login page.

* US2: Test Login fail. Verify the unsuccessful login shows the error message: "Invalid credentials or access not granted due to user account status or an existing user session."

* US3: Test Update Profile. Verify that changing the first name is visible in the Welcome text.

* US4: Test a checking deposit transaction. Verify that the new deposit amount transfer is visible in the checking list and it changed the balance.

* US5: Test a savings deposit transaction. Verify that the new deposit amount transfer is visible in the savings list and it changed the balance.
